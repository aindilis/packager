Annotation

Annotation of Language Resources summarizes at least some of the various tools and methodologies we might pursue in creating these recognizers.

Alembic - We will use one of several methods for automatically extracting information from webpages concerning aspects of applications. We will be able to automatically recognize package names, author names, author emails, package homepages, package mirrors, package versions, packages documentation, manuals, copyright/license information, operating system, and so on. To accomplish this we will possibly use several different freely available tools. Note that Gate, Alembic and the BBN Tagger will work here.
