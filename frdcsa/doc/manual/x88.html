Auditing of Sources

A primary RADAR feature is the ability to track sources over time. This would include the following capabilities. Relate extracted metadata to source document. Monitor source documents for changes. Cache copies of different source documents (a.l.a. wayback machine) Track new, continuous, and removed content across copies.
