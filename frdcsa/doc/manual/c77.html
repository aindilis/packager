Emacs Interface

The RADAR minor mode is the primary interface with Emacs.

Reuse of existing Emacs tools.

Existing Emacs functions must be thoroughly utilized to avoid duplicate work. However, modular insertion of proper Emacs methodologies as they are discovered and formalized will also work. For instance, writing a NER extractor for the desired package metadata is useful, but certainly, existing Emacs tools for identifying e-mail, should be used.
